## Topic Summary
GitHub Copilot, Cursor, and similar AI coding assistants have become standard tools in
professional software development almost overnight — but their rise has triggered a
genuine industry debate: are they boosting developer productivity while quietly shrinking
the entry-level job market for new graduates like us? This project covers current adoption
data, the real (vs. hyped) productivity gains, the contested junior-hiring debate, and what
it means for CS students entering the field.
